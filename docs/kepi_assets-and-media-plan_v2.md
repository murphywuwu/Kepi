## 1. 素材总原则

- 素材优先服务主流程，不做无关装饰。
- 先做能跑通一局的核心素材，再补可选表现层。
- 所有素材都要适配网页端和黑客松 demo 节奏。
- 断网可玩，因此关键素材必须本地可用。
- 结局相关素材需要支持本地兜底，不依赖在线资源。
- V2.0 素材优先服务 10 分钟微型局的三件事：典当痛感、水客危机、土楼反哺。
- 美术风格按资产类型拆开执行：角色 / 敌人采用 V2 Prompt 包的白底全身赛璐珞平涂；场景采用 V2 Prompt 包的国漫动画电影场景，大色块平涂并可保留薄雾空气感；UI 保持米纸木框的低饱和手工感。
- 角色、敌人、道具、场景分开出图，不把 UI 文本、角色名、费用、标题直接画进图片。

## 1.0A 风格偏差复盘与 V2 修正

V2 Prompt 包是在教书先生 / 水客实跑后定稿的修正版。核心结论：角色要像《大鱼海棠》的感觉，不能把质感写成“水彩厚涂 + 薄雾”，而要写成“动画赛璐珞平涂”。

偏差原因：

- `水彩晕染、厚涂笔触、插画质感、雾气、毛边` 会把角色拉向朦胧、脏灰、偏写实的插画，不稳定。
- `精品概念设定图、细腻材质、真实皮肤高光、体积光` 会让脸变成熟、服装变复杂，偏离白底棋子立绘。
- 一步到位同时要求“客家服饰 + 大鱼海棠质感”容易翻车：要么服装像日式古风 / 和服，要么画风像水彩插画。
- 即梦对“客家”识别不足，必须把客家服装的对襟 / 大襟斜襟、立领 / 小圆领、一排布盘扣、纯色靛青短衫 / 长衫写死。

修正口径：

- 角色 / 敌人统一追加 V2 通用画风行：`《大鱼海棠》国漫动画赛璐珞平涂,大色块纯色平铺,阴影只用简洁的硬边色块,描边干净清晰`。
- 角色脸统一写：`面部简洁圆润大眼+深棕瞳孔柔和`，配合 `纯白背景,全身立绘,完整无裁切,无阴影投影`。
- 角色统一走两步法：第一步文生图先出客家形象；第二步图生图只转质感，并锁死同样的脸、发型、服装道具、站姿。
- 场景可以保留薄雾空气感，但角色 / 敌人 / 道具禁止水彩晕染、厚涂笔触、雾气和毛边。
- 新角色先单张试跑并与教书先生并排验收；通过后再替换同组资产。

## 1.0B V2 角色画风总结（白底棋子立绘）

角色最终风格概括为：**大鱼海棠国漫动画赛璐珞平涂 + 客家服饰写死 + 白底全身棋子立绘**。

- **整体观感**：干净、明亮、简约，像动画片里的一帧，也像可直接抠进棋盘的角色设定稿。
- **线条**：描边干净清晰，衣服边缘利落不晕染；内部褶皱少，靠硬边阴影分块。
- **上色**：大色块纯色平铺，阴影只用简洁硬边色块；不要水彩晕染、厚涂笔触和插画质感。
- **脸部**：面部简洁圆润，大眼，深棕瞳孔柔和，眉鼻嘴克制；少年 / 青年角色要有清爽少年感，年长角色也保持干净圆润，不堆皮肤纹理。
- **服装**：必须是客家衫逻辑，不能被模型脑补成和服、日式交领、飘逸古装或复杂花纹镶边。
- **色彩**：低饱和暖色调，靛青 / 深蓝 / 灰黑为主，米白、赭石、暗红点缀；干净通透，不灰不脏。
- **背景**：纯白背景，全身立绘，完整无裁切，无阴影投影，无文字水印。
- **道具**：道具一眼说明身份，但不横向铺满画面；水客的木箱中国结 + 竹篓侨批是身份命门。

## 1.0C 角色生产两步法（强制）

所有我方角色统一走两步法：

1. **第一步 · 出形象**：文生图先解决“长相对不对、客家不客家”。Prompt 必须写死客家服装要点，不强求最终质感一次到位。
2. **第二步 · 转质感**：拿第一步满意图做图生图，只把绘画质感转成《大鱼海棠》动画赛璐珞平涂，形象全锁死不动。

第二步通用模板：

```text
【上传第一步的图】保持画面中人物的一切不变——同样的脸、发型、{服装道具清单}、站姿全部保留,
只把绘画质感转换成《大鱼海棠》动画电影的赛璐珞平涂风格:
衣服改成大色块纯色平铺、阴影只用简洁的硬边色块、去掉所有笔触感和水彩纹理,
描边干净清晰、颜色干净通透,像动画片里的一帧,简约不繁复,
低饱和暖调,纯白背景,全身立绘,完整无裁切,
负向:水彩,厚涂笔触,插画质感,雾气,毛边,颜色脏,过度写实
```

客家服装硬约束：

- 对襟或大襟斜襟 + 一排布盘扣，优先写 `5颗一字布扣`。
- 立领或小圆领，不写和服敞开的交领。
- 纯色靛青 / 深蓝 / 灰黑，无花纹无镶边，布料质朴厚实。
- 男装下装：大裆裤 + 绑腿 + 布鞋；劳作角色可配草鞋。
- 劳作角色（农夫 / 守卫 / 水客）穿短衫；体面角色（教书先生 / 风水先生 / 族长 / 乡贤）穿长衫。
- 有客家服装参考图时，第一步就喂参考图；第二步只转质感，不改版型。

## 1.1 风格基线

- 场景：东方国漫动画电影场景，大色块平涂、干净清晰，参考《大鱼海棠》的温润气质；场景允许薄雾空气感和柔和体积光，但必须保留客家土楼的圆楼、天井、木回廊识别度。
- 人物：V2 Prompt 包的白底全身动画赛璐珞平涂；干净圆润脸、大眼深棕瞳、低饱和暖色、干净描边、大色块纯色平铺、简洁硬边阴影；不做水彩晕染、厚涂笔触、真实皮肤纹理或复杂衣料渲染。
- 我军：蓝靛土布、客家大襟衫、布扣、腰带、斗笠、凉帽、书卷、罗盘等文化识别物，整体温暖有人情味。
- 敌军：与我军同一笔触体系，但更冷硬、器物成精、青灰与朱红压迫感更强。
- 土楼：圆形土楼是主识别点，四状态必须是同一机位、同一构图、同一个家逐渐变好。
- 画面：不写实、不 3D、不走旧动画脏灰校色，不做过度 Q 版、西式卡通或泛日式古风。

### 1.2 最终视觉确认（2026-06-30）

- 所有图片资源都按正式素材产出，不做占位图。
- 角色与敌人默认输出为白底全身立绘，便于先锁画风与造型；进入游戏资产时再按需要抠透明底。
- 角色 / 敌人默认尺寸：`1024x1280`，3/4 正面，居中，完整无裁切。
- 场景默认尺寸：`1600x900`，横版大图，保留顶部与底部 UI 安全区。
- UI 图标默认尺寸：`256x256`，统一圆徽章或轻徽章边框 + 中心符号。
- 主背景默认态：修缮态；破败态与翻新态共用同一视角和构图模板。
- 土楼阶段按 V2.0 `homeRepairTier` 处理：0 破败、33% 修缮·初见、66% 翻新·同心、99% 焕然·不屈。
- 主场景气氛：春末到初夏的清晨薄雾 / 晨光，左上暖光、右下阴影，保留海雾和潮气但不铺满海面。
- 整体风格：角色 / 敌人以白底赛璐珞平涂设定图为主，场景以国漫动画电影场景为主；两者共享靛青、米白、暖赭、青灰、暗红的低饱和色彩体系，但角色禁水彩雾气，场景可保留空气感薄雾。
- 角色面部：脸永远干净圆润，大眼深棕瞳，眉鼻嘴克制；年少角色要有少年感，年长角色也保持动画化的圆润清爽，不堆皱纹、胡茬、皮肤纹理和碎线。
- 结局气氛：风浪、散信、船影、手势高光分三层，不做灾难片。
- 所有图片内不加角色名、费用、标题、解释、UI 文案或水印。

### 1.3 V2.0 机制口径

- 一局固定 4 回合，约 10 分钟；素材不再围绕 6 关长局铺排。
- 客批阈值为 4；归乡票由 4 封客批显化，不再按 5 封客批设计。
- 水客与乡贤是可上场棋子，不再是无敌后勤挂件；水客必须有“脆弱但重要”的视觉识别。
- 水客胜利且存活才产出 `+1 客批 +20 桑梓值`；胜利但水客死亡要有“赢了但信没了”的结算反馈。
- 典当是核心经济抉择：消耗 1 封客批，获得 15 金币，需要独立按钮、燃烧/撕裂动效与沉重音效。
- 第 4 关械斗火必出，作为刺客跳后排并优先锁水客；素材需要支持刺客预警圈与跳跃特效。
- 土楼庇护三档 Buff：33% 生命护盾、66% 攻速提升、99% 免死一次；素材需要对应图标、特效和阶段过场。
- 每回合结算展示 AI / fallback 侨批旁白；素材需要给旁白卡片、署名、小纸条纹理留位置。

### 2.2A 出图规则

- 我军统一走 V2 Prompt 包风格：`《大鱼海棠》国漫动画赛璐珞平涂`、白底全身、大色块纯色平铺、简洁硬边阴影、干净清晰描边、面部简洁圆润大眼 + 深棕瞳孔。
- 角色第一关键词不再写“水彩厚涂、插画质感、雾气、体积光、精品概念图、轻二次元写实、自然皮肤高光、细腻材质”；这些词会把角色拉回偏写实或脏灰。
- 每个我军角色都要保留少年感 / 青年动画感：干净圆润脸、大眼深棕瞳、清爽比例、表情明亮克制。族长、风水先生等年龄身份可以保留，但脸部和上色仍要动画化、干净，不堆真实皱纹。
- 服装必须写死客家服饰结构：对襟或大襟斜襟 + 一排布盘扣、立领或小圆领、纯色靛青 / 深蓝 / 灰黑、无花纹无镶边。
- 劳作角色（农夫 / 守卫 / 水客）穿短衫；体面角色（教书先生 / 风水先生 / 族长 / 乡贤）穿长衫。
- 服装保持简化，复杂度放在衣长、头饰、必要道具和服饰层级；道具不能大到抢画面。
- 族长、风水先生、农夫、守卫的服饰层级要一眼可辨。
- 水客强调“挑担收信 + 脆弱但重要”，必须像需要保护的少年 / 青年上场棋子，而不是安全 NPC 或中年路人。
- 乡贤强调“修家园的体面感 + 低攻击远程辅助”，必须能和普通战斗棋子同屏站位。
- 敌军统一走“同源干净描边 + 赛璐珞平涂 + 冷硬器物成精”的反派感，不能变成粗重怪兽概念图或写实怪物设定。
- 敌人必须让人看出它阻挡的是哪段返乡历史。
- 每次生成新角色时，先生成 1 张第一步白底图验收客家形象；形象通过后再用第二步图生图转大鱼海棠赛璐珞平涂。先看脸是否干净圆润、大眼深棕瞳、少年感是否成立，再看客家服装、盘扣、道具是否准确。

### 2.2E 出图建议

- 角色先跑白底版本确认造型和画风，再批量做透明底或抠图。
- 每个新角色先只跑 1 张给用户确认，不通过就只改 STYLE 锚，不继续批量生成。
- 通过单张确认后，每个角色至少跑 3 张挑选，尤其是族长的瓜皮帽、五颗盘扣、风水先生的罗盘和八卦纹容易丢。
- 如果角色看起来像泛日系少年、普通动漫脸、写实插画、旧动画、民俗插画或灰黄厚涂，说明 STYLE 锚仍被错误脸部词 / 场景词 / 写实词污染，应强化“V2 赛璐珞平涂、干净圆润脸、大眼深棕瞳、纯白背景、硬边阴影”，并删除“精品概念、细腻材质、真实皮肤、体积光、厚涂”等角色 prompt 关键词。
- 敌人若过于可爱，加重“压迫、冷硬、诡异、menacing、反派氛围”。
- 敌人若过于写实，加重“干净描边、赛璐珞平涂、大色块、硬边阴影、白底动画设定图、概括不琐碎”。
- 产出文件命名见 [文档模板与命名规范 v1](kepi_document-conventions_v1.md)。

#### 2.3A UI 统一规则

- 图标统一采用圆徽章或轻徽章边框，中心放核心符号，不做纯文字或写实物件照。
- 图标默认尺寸为 `256x256`，透明底，便于前端缩放和拼接。
- 主色以暖木褐、米宣纸、靛蓝和少量琥珀金为主。
- 图标需要能独立识别，不能只靠按钮文字解释。
- 图标与角色一样不加字，不加说明标签。
- UI 不追求灰黄旧动画感，仍然要干净、可读、适合网页游戏。

## 6. 资源规格建议

### 6.1 图片

- 建议使用 `PNG` 或 `WebP`。
- 角色 / 敌人出图阶段默认白底全身立绘，默认 `1024x1280`。
- 进入游戏前按需要抠透明底，透明底版本仍保存到 `public/images/characters/` 或 `public/images/enemies/`。
- 棋盘和背景图优先横向大图，默认 `1600x900`。
- UI 图标默认 `256x256`。
- 角色 / 敌人全身立绘，主背景单独出图，不混在同一张里。
- UI 图标采用统一圆徽章风格。

### 6.2 音频

- 建议使用 `MP3` 或 `WAV`。
- BGM 和循环环境音要控制体积。
- 客家话朗读尽量按信件拆条。

### 6.3 命名建议

```txt
public/
  images/
    board/
    characters/
    enemies/
    cinematics/
    ui/
    ending/
    fx/
  audio/
    bgm/
    sfx/
    voice/
```

- `characters/` 与 `enemies/` 使用最终透明底正式立绘。
- `board/` 与 `ending/` 使用横版大图。
- `cinematics/` 使用横版过场大图与透明底前景素材，命名优先 `kepi_victory-*`。
- `ui/` 同时容纳图标与纹理，统一前缀 `kepi_ui_*`。
- `fx/` 放典当燃烧、水客危机、土楼 Buff、刺客跳跃等透明底或序列帧素材。

## 7. 降级策略

- 土楼四状态吃紧时，可降成两张主图：破败态 + 焕然·不屈；33/66 阈值用金色粒子、护盾图标和 HUD 文案补足。
- 典当动效吃紧时，先用静态半燃信封 + 印章音效 + 金币数值跳变，不阻断机制表达。
- 水客危机动效吃紧时，先做 HUD 红边 / 黄边和结算文案，不强求角色死亡动画。
- 刺客跳跃动效吃紧时，先用第 4 关红色预警圈 + 瞬移残影表达。
- 棋子 / 敌人立绘不足时，可先用同风格白底立绘裁切入场，不急于做透明底精修。
- 结局信件不足时，先保证真实侨批原文优先。
- 朗读音频不足时，先保证关键结局信有声音。
- 视觉特效不足时，不影响主流程。
- UI 纹理未齐时，不使用明显现代感的 glossy 或 3D 资源，临时 CSS 也必须保持米纸、木框、低饱和。
